# ceki.db
fast json database module


modülü kullanmak için:

```js
import cekiDB from "./index.js";
```

olarak modülü tanımlayın daha sonra

```js
cekiDB.get()

cekiDB.set()

cekiDB.delete()

cekiDB.add()

cekiDB.subtrack()

cekiDB.push()
```

methodlarını kullanabilirsiniz.


ES modül tipini kullanan arkadaşlar [Burada](https://github.com/jahkyxd/ceki.db/tree/ceki.db-ES) ona uygun projeyi indirebilirler.
